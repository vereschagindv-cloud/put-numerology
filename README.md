# Путь — нумерология по дате рождения

Прототип мини-сайта: расчёт числа жизненного пути и матрицы судьбы (квадрат Пифагора) по дате рождения, с расширенным платным разбором (день/месяц/год по сферам жизни) за символическую плату.

Полностью статичный, без сервера — все расчёты выполняются в браузере. Оплата в демо-режиме эмулируется.

Открыть: `index.html`.
