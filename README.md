# Путь — нумерология по дате рождения

Мини-сайт: расчёт числа жизненного пути и матрицы судьбы (квадрат Пифагора) по дате рождения, с расширенным платным разбором (день/месяц/год по сферам жизни) за символическую плату.

Полностью статичный фронтенд, все расчёты выполняются в браузере. `privacy.html` и `offer.html` — политика конфиденциальности и пользовательское соглашение.

## Статус приёма платежей и почты

Кнопка оплаты и отправка на почту пока не подключены к реальному провайдеру/email-API — на сайте это явно указано (в `.unlock-note` и в футере). Перед реальным запуском нужно:

1. Оформить статус самозанятого/ИП и завести аккаунт в платёжном провайдере (ЮKassa/CloudPayments).
2. Поднять бэкенд (например, Cloudflare Pages Functions или Vercel) для проверки вебхука оплаты и отправки email через провайдера (Resend/SendGrid/Unisender).
3. Дополнить `offer.html` реальными реквизитами исполнителя.
4. Убрать формулировки про «в разработке» из `index.html`, когда оплата и почта заработают по-настоящему.

Открыть: `index.html`.
